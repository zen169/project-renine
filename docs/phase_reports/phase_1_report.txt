============================================================
MAJOR DECISIONS
============================================================
1. PermissionLevel enum lives in tools/permissions.py as the
   single source of truth. All other modules import from there.

2. Electron UI uses vanilla JS/TS for Phase 1 without a React
   build pipeline. Components are structured as classes that
   will migrate to React components in later phases.

3. ContextSanitizer is wired in from Phase 1 even though only
   local Ollama is used. This prevents any future external
   API path from bypassing sanitization.

4. STT model uses lazy loading to avoid blocking app startup.
   Module-level caching prevents repeated GPU allocation.

5. All config values come from YAML files with env var
   interpolation — no hardcoded values in source code.
