============================================================
MAJOR DECISIONS
============================================================
1. PermissionLevel lives in tools/permissions.py — single source
   of truth. All modules import from there.

2. Electron UI uses vanilla TS classes for Phase 1 without a React
   build pipeline. Component structure mirrors future React shape.

3. ContextSanitizer wired in Phase 1 even though only local Ollama
   is used — prevents any external API path bypassing sanitization.

4. STT model uses lazy loading — avoids blocking app startup.
   Module-level caching prevents repeated GPU allocation.

5. All config from YAML files with ${ENV:-default} interpolation
   — no hardcoded values, no secrets in source.

6. Session handoff system established — project is documentation-
   driven. Chat history is temporary; docs are permanent.
