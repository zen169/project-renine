ARCHITECTURE & DESIGN DECISIONS
============================================================
1. Hardened Path Traversal & Directory Whitelisting:
   `validate_path` in `input_validator.py` was refactored to resolve absolute paths using `Path.resolve()` and verify they reside within whitelisted directories configured in `config/security.yaml`. Any attempt to access outside the whitelist or paths matching blocked definitions raises an `InputValidationError`.

2. Native, Dependency-Free Windows API Integration:
   - Volume control uses inline C# loading standard COM APIs (`IAudioEndpointVolume`) executed via a subprocess PowerShell script, bypassing dependency heavy modules like `pycaw`.
   - Clipboard integration uses Win32 API functions (`OpenClipboard`, `GetClipboardData`, etc.) loaded directly via `ctypes` for clean memory allocations and no external dependencies.

3. Cached File Caching (file_index):
   To avoid continuous disk scanning, FileAgent creates and queries a `file_index` table in Layer 3 `mind.db`. Directory indexing is performed incrementally, updating metadata and generating previews (first 500 characters of text files) only when file size or modification time changes.

4. Spreadsheet Analysis using Pandas:
   SpreadsheetAgent parses CSV and Excel spreadsheets using `pandas` and `openpyxl`. Column lists and summary statistics are computed locally. For general questions, the computed statistics and top rows are forwarded to Ollama, enabling offline natural language query answers.
